# 🏥 ระบบบันทึกข้อมูลการตรวจสุขภาพ (Health Check-up Digital System)
ระบบบันทึกและออกรายงานผลการตรวจสุขภาพเบื้องต้น พัฒนาด้วย **Google Apps Script**

## 🌟 จุดเด่นของโครงการ
- **Automation:** สร้างรายงาน PDF จาก Google Docs อัตโนมัติทันทีที่กรอกข้อมูล
- **Paperless:** ลดการใช้เอกสารกระดาษ 100% และจัดเก็บข้อมูลบนระบบ Cloud
- **Efficiency:** ลดระยะเวลาการทำงานของเจ้าหน้าที่ในการคีย์ข้อมูลย้อนหลัง

## 🛠️ เครื่องมือที่ใช้ (Tech Stack)
- **Language:** Google Apps Script (JavaScript based), HTML
- **Platform:** Google Sheets, Google Forms, Google Drive
- **Template:** Google Docs (สำหรับสร้างรายงาน PDF)
